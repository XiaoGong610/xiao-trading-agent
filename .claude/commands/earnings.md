Summarize the most recent earnings call for: $ARGUMENTS

Use web search to find earnings call details, transcripts, and analyst coverage. Cover:

## Results vs. Expectations
- Revenue: actual vs. estimate, YoY growth
- EPS: actual vs. estimate, YoY growth
- Beat/miss on key segment metrics

## Management Commentary
- Key quotes and tone (confident, cautious, defensive?)
- Strategic priorities highlighted
- Any new initiatives, products, or pivots mentioned

## Guidance
- Next quarter guidance vs. consensus
- Full year guidance (raised, maintained, lowered?)
- Any notable changes from prior guidance

## Analyst Q&A Highlights
- Toughest questions asked and how management responded
- Any evasiveness or surprising candor on specific topics

## Market Reaction
- Stock move after earnings (AH and next day)
- Analyst rating/target changes following the call

## Key Takeaways
- 3-5 bullet points: the most important things an investor should take away
- What changed about the thesis (if anything)?
- Bullish or bearish signal overall?

Save the output to `research/$ARGUMENTS.md` (use uppercase ticker as filename).
Start the entry with a date separator: `---` followed by `# TICKER — Earnings Call | YYYY-MM-DD`.
If the file already exists, prepend the new analysis above all previous entries (after the file title). Never remove historical entries.
