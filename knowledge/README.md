# Knowledge Base

Reference docs for trading decision-making. Skills consult these for context and nuance.

## Structure

```
knowledge/
  sectors/       — Sector-specific logic, metrics, cycle dynamics
  strategies/    — When to use each strategy, rules, edge cases
  frameworks/    — Reusable mental models and analytical frameworks
  signals/       — How to interpret technical and options signals
```

## How to Use

- Skills reference these files when making recommendations
- Updated as we learn from trades and refine our thinking
- Not stock-specific — these are general principles that apply across stocks
